Лабораторна робота №6: Об'єктно-орієнтоване програмування в Java📝 Опис проєктуОтримати практичні навички роботи з принципами об'єктно-орієнтованого програмування (ООП) на мові Java: проектування класів, інкапсуляція даних через конструктори, перевантаження методів, реалізація спадкування, перевизначення методів (@Override) та застосування абстрактних класів.🛠 ВимогиJava Development Kit (JDK): версія 17 або вище (рекомендовано 25)Консоль/Термінал: для компіляції та запуску коду📂 Архітектураsrc/├── Person.java           # Клас, що описує людину та її базові дії├── Phone.java            # Моделювання роботи телефону (перевантаження методів, varargs)├── StudentDemo.java      # Успішність студентів та магістрів (спадкування, перевизначення)├── ShapeHierarchy.java   # Ієрархія тривимірних геометричних тіл для обчислення об'єму└── CarDemo.java          # Абстрактний клас транспортного засобу та його конкретні реалізаціїКожен файл є самостійною програмою (містить власний метод main), яка наочно демонструє конкретний аспект об'єктно-орієнтованої парадигми.📋 Завдання1. Клас Person (Створення об'єктів та конструктори):Створити клас Person із полями fullName та age.Реалізувати два методи: move() та talk(), які виводять інформацію про поточні дії людини в консоль.Додати два конструктори: за замовчуванням (дефолтний) та повний (параметризований).Ініціалізувати два об'єкти різними конструкторами та викликати їхні методи.2. Клас Phone (Перевантаження методів та аргументи змінної довжини):Створити клас Phone із характеристиками: number, model, weight.Оголосити три конструктори: повний, з двома параметрами (number, model) та порожній.Реалізувати метод receiveCall(String name) та перевантажити його версією receiveCall(String name, String number).Створити метод getNumber() для повернення номера поточного телефону.Реалізувати метод sendMessage(String... numbers), який приймає аргументи змінної довжини (varargs) і виводить список номерів для розсилки.3. Клас StudentDemo (Спадкування та розширення логіки):Створити базовий клас StudentProgress для збереження імені, прізвища та оцінок з трьох предметів (Математика, Фізика, Java).Реалізувати методи введення даних setData(), виведення printInfo() та розрахунку середнього бала average().Створити похідний клас MasterStudent, що розширює базовий новими полями: тема наукової роботи (researchTopic) та науковий керівник (mentor).Перевизначити метод printInfo() для виведення специфічних даних магістра через виклик super.printInfo().4. Клас ShapeHierarchy (Ієрархія класів та обчислення об'ємів):Проектувати базовий клас Shape із властивістю volume (об'єм).Створити проміжний клас SolidOfRevolution (тіло обертання), що додає поле radius.Реалізувати кінцеві класи-спадкоємці: Ball (куля) та Cylinder (циліндр) з автоматичним обчисленням об'єму за математичними формулами при ініціалізації.Створити окрему гілку спадкування для класу Pyramid (піраміда), що вираховує об'єм на основі площі основи та висоти.5. Клас CarDemo (Абстракція та поліморфізм):Реалізувати абстрактний клас Car зі спільними характеристиками (model, color, maxSpeed) та готовим методом gas().Визначити абстрактний метод abstract void brake();, який не має власної реалізації.Створити підкласи Sedan та Truck, які обов'язково перевизначають метод brake(), реалізуючи специфічну поведінку гальмування для кожного типу автомобіля.🚀 ЗапускКомпіляціяПерейдіть у терміналі до каталогу вашого проєкту (де знаходиться папка src) та скомпілюйте всі файли лабораторної роботи в бінарну папку bin:Bashcd Lab6javac -d bin src/*.javaВиконанняОскільки кожна частина лабораторної роботи містить власну точку входу (main), ви можете протестувати та запустити будь-який демонстраційний клас окремо:Bashjava -cp bin Personjava -cp bin Phonejava -cp bin StudentDemojava -cp bin ShapeHierarchyjava -cp bin CarDemo
